Nesse repositório tem todas as etapas de desenvolvimento do jogo. 
Os arquivos com nome CUTSCENE - São os arquivos com os códigos que representam as cenas do jogo.
Os arquivos com nome CAPÍTULO - São os arquivos com os códigos que representam somente as gameplays de cada parte do jogo. 
Os arquivos com nome PARTE - São os arquivos com os códigos que representam a uniam das cutscenes com a gameplay de cada parte do jogo. 

Não há, por enquanto, um único arquivo que reuna todos os 3 capítulos em sequência.
